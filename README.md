## 常见问题 FAQ

**Q：搬瓦工 JMS 和搬瓦工 VPS 有什么关系？**

同属 IT7 Networks 旗下。搬瓦工 VPS 是你自己租一台服务器，需要手动配置节点；JMS 是他们直接把这层工作做好了，你买套餐直接拿订阅链接用。两者面向不同需求的人群。

**Q：JMS 的流量是按单向还是双向计算？**

多数套餐按双向计算，即上传和下载都消耗流量配额。每个账单周期流量会重置（具体日期以购买时的邮件为准，一般是购买日）。LA 系列套餐里有一个 S801 下载专用节点，只计算十分之一的流量，适合大文件下载的场景。

**Q：JMS 支持哪些客户端？**

Shadowsocks 和 V2Ray 协议均支持，主流的客户端软件（iOS/Android/Windows/macOS 均有对应软件）都能导入 JMS 的订阅地址直接用，不需要手动填节点信息。

**Q：搬瓦工 JMS 价格会变吗？**

历史上调过几次价，整体趋势是缓慢上涨，但涨幅不大。续费价格和首次购买相同，不存在续费涨价问题。

**Q：7 天退款是怎么申请的？**

登录 JMS 账号后台，通过工单（Open Ticket）提交退款申请，说明原因，通常在 1-2 个工作日内处理，退款原路返回到你的付款账户。
